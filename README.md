# Islamy App

Islamy is a comprehensive mobile app designed for Muslims, offering a rich array of features to enhance their spiritual journey. The app includes the Quran, Ahadith, a live radio, and a digital Sebha for counting prayers. With support for dark mode, Islamy ensures a comfortable user experience in low-light environments. Additionally, it offers localization options, making it accessible to a global audience. Islamy is the perfect companion for daily worship and spiritual growth
